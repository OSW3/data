# Geo : Countries

JSON Array of countries, alpha codes and numeric code.

- country : The english name of the country.
- alpha_2 : 2 chars country alpha code.
- alpha_3 : 3 chars country alpha code.
- un_code : numeric code.

```json
// ...
{
    "country": "France",
    "alpha_2": "FR",
    "alpha_3": "FRA",
    "un_code": "250"
},
// ...
```
